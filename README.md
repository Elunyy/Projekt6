# Projektbeskrivning
## Postvanor
En websida som visualiserar postavlämning i en brevlåda med hjälp av en sensor. Sensorn ska känna av vilken dag och tidpunkt posten levererades. Websidan kan beskriva postvanor; vilka dagar du får post och ungefär vilken tid på dagen du brukar få posten, hur lång tid det tar innan du hämtar in posten eller andra avvikande data som om du inte fick post idag även fast du brukar få det.

**Målgruppen** är de som har långt till sin brevlåda, eller svårare att ta sig dit och inte vill gå i onödan. 
**Syftet** är att underlätta posthämtning och att skapa en behagligare upplevelse att hämta posten. Personer som har rörelsesvårigheter och behöver gå ut en längre bit för att hämta posten kan finna sig i en situation där posten inte kommit än. För att undvika detta kan personen först kolla på websidan för att se om posten kommit eller ej. Även personer utan funktionsnedsättningar kan dra nytta ur dessa funktioner om man inte har tid för att gå och kolla om man fått brev eller om det är kallt ute och man helst vill undvika att gå ut.

## Möjliga funktioner
* Ja/nej, finns det något i brevlådan?
* Hur låg tid tar det för dig att hämta posten?
* Vilka dagar lämnas post?
*	Vilken tid lämnas post på dagen?
* Påminnelse att hämta posten.
*	Koppla mail för uppdateringar.
* Heat-map.
* Avvikande aktivitet.
